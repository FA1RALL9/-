# Сайт записи (Flask + SQLite)

Функции:

- Автоматически показывает **сегодняшний день недели** и время встречи из настроек
- **Ограничение мест**: при записи уменьшается количество доступных мест, в минус не уходит, при 0 показывается “Мест больше нет”
- Все записи сохраняются в **SQLite**
- На следующий день записи за прошлые даты **автоматически удаляются** (очистка запускается при открытии страниц)
- **Админ‑панель**: менять время и лимит мест по дням недели, смотреть/удалять заявки за сегодня

## Быстрый запуск (Windows)

1) Создать и активировать виртуальное окружение:

```bash
python -m venv .venv
.venv\Scripts\activate
```

2) Установить зависимости:

```bash
pip install -r requirements.txt
```

3) (Рекомендуется) задать логин/пароль админа:

```bash
setx ADMIN_USER admin
setx ADMIN_PASSWORD "смените_меня"
```

Перезапустите терминал после `setx`, чтобы переменные применились.

Если переменные не заданы, по умолчанию вход в админку: **admin / admin**.

4) Запуск:

```bash
python app.py
```

Открыть:

- Главная: `http://127.0.0.1:5000/`
- Админка: `http://127.0.0.1:5000/admin/`

## Хранение данных

- База данных: `data/app.db`
